CHARACTER SPRITES

To have a playable game, we need an avatar for the user to manipulate. And so, the first task on the list is to create a character sprite.

In Godot, every element that interacts with one another is called a node. To create our character sprite, we first have to create a Node2D, and load in the character sheet that can be downloaded through the asset pack. The character sheet has multiple views of a character avatar, which we will later be able to manipulate in order to create character animations.

When we drag the sprite sheet into the new node that we created, you will notice that it displays 16 different character sprites, facing different directions and in different poses. In order to have the sprite sheet display only one character, go to the Animations tab on the right and find the categories for Hframe and Vframe. Hframe and Vframe stands for horizontal frame and vertical frame respectively. Because there are four rows and four columns of the character sprite, we input the number 4 into both categories. After doing this, it should only display the first frame.

We should also create a new node for any other characters that might be present for this game. The asset pack also includes a character sheet for a cow, and that may be a fun NPC to display, so we would make a new node for the cow character as well.
