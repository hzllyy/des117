COLLISION SHAPES

Finally, we want to work on collision shapes. In order for a character to interact with the objects around them, we need to make sure that our avatar isn't just walking all over everything. So, I will be demonstrating how to create a collision shape-- in this case, it will be in the form of a "higher ground" that the character sprite cannot walk onto unless it's through a certain path.

I'll start by going into the tile set category and selecting the "paint" option. This will bring you to a window that displays the selected tile, as well as a couple of key toggles that allow you to manipulate a highlighted blue area.

The blue highlighted area indicates the part of the tile you would like to make into a collision shape. In the case of this tile, I wanted it so that the character sprite wouldn't be able to move onto the tile if approaching it from the left side. Therefore, I adjusted the highlighted area to block off the tile from the left.

Now, when you run the game, you will see that the character is unable to move onto the elevated surface. However, it can go up on the surface when there is a means of entry, like the set of stairs I provided.

With everything we've gone over, you should now be able to build a very simple and beginner-friendly scene of your own in Godot!
