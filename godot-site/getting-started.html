LEARNING GODOT

To get started, we first need to download Godot. I am using version 4.2.2., which has a lot of new updates that simplifies the complexity of game development. You can download the latest version by visiting Godot's official site.

Something that reduces the complexity of this tutorial is downloading asset pack. An asset pack includes all the elements we need to build a simple scene, including pre-customized character sprites and tile maps.

The asset pack I chose to use is the Sprout Lands Asset Pack by Cup Noodle; you can download the same asset pack for free here. This asset pack will give us access to pre-created character sprites, tilemap building tools, and objects!
