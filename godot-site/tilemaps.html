BUILDING TILEMAPS

It wouldn't be a game if there wasn't a world to roam and explore! To get started on the tilemap, we have to create a Tile Map child node. Once this is created, we can import the tile set provided in the asset pack and create a new tile set, and this will give us the tools we need to begin building our map.

Building a tile map is simple from here on out. We can select whichever tile we want to put into our scene, and then we can just simply left click the tiles wherever we want them.
